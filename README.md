\# Albion Profit Maker



A free web application for Albion Online players to maximize silver earnings through comprehensive market analysis.



\## Features



\- \*\*Crafting Profit Calculator\*\* - Calculate profitable crafting opportunities

\- \*\*Market Flipping Finder\*\* - Discover cross-market arbitrage opportunities  

\- \*\*Real-time Price Checker\*\* - Compare prices across all cities

\- \*\*Farming Calculator\*\* - Optimize resource gathering and island farming

\- \*\*Data Persistence\*\* - Save your favorite opportunities and settings



\## Live Demo



Visit: \[ali926.github.io/albion-profit-maker](https://ali926.github.io/albion-profit-maker)



\## Local Development



1\. Clone this repository

2\. Open `index.html` in your web browser

3\. No server required - it's a static web app!



\## Technology Stack



\- HTML5, CSS3, JavaScript (ES6+)

\- Albion Online Data Project API

\- Local Storage for data persistence

\- Responsive design for mobile and desktop
